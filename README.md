# Rock Paper Scissors Game

A fun and interactive Rock Paper Scissors game built using HTML, CSS, and JavaScript.
This project demonstrates fundamental front-end development concepts, including DOM manipulation, event handling, and basic game logic.

# Features
* Single-player game (User vs Computer)

* Random computer move generation

* Instant result display (Win / Lose / Draw)

* Play again without page reload

* Responsive and user-friendly UI

* Lightweight & fast (no external libraries)

# Technologies Used
* HTML – Structure of the game

* CSS – Styling and layout

* JavaScript – Game logic and interactivity

# Game Rules
1. The user selects Rock, Paper, or Scissors.

2. The computer randomly chooses one option.

3. Winner is decided based on standard rules:

4. Rock beats Scissors

5. Scissors beats Paper

6. Paper beats Rock

7. Same choices result in a draw.
